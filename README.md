# Prime Number Checker (Python)

## Description
This project is a simple Prime Number Checker built using Python. The program takes a number as input from the user and determines whether it is a prime number or not. It uses basic looping and conditional logic to perform the check and allows the user to test multiple numbers until choosing to exit.

## Features
- Checks whether a number is prime or not
- Input validation for non-numeric values
- Allows multiple checks in one run
- Simple command-line interface

## Technologies Used
- Python
